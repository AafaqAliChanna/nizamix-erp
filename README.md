# BAAZ GRAMMER SCHOOL - ERP System

A professional, multi-page Enterprise Resource Planning (ERP) system designed specifically for Pakistani schools. Built with modern web technologies for managing students, staff, attendance, fees, academics, and comprehensive reporting.

## Features

### 📊 **Dashboard**
- Real-time statistics and key metrics
- Quick-access cards for all major modules
- Beautiful data visualization with key performance indicators

### 👨‍🎓 **Student Management**
- Complete student records and profiles
- Roll number tracking
- Class assignment management
- Student status monitoring
- Quick add/edit functionality

### 📋 **Attendance Tracking**
- Daily attendance marking
- Class-wise attendance records
- Attendance percentage calculations
- Leave and absence tracking
- Real-time attendance reports

### 💳 **Fee Management**
- Monthly fee tracking
- Payment recording and verification
- Fee payment status (Paid, Pending, Partial)
- Outstanding balance calculation
- Financial reports

### 📚 **Academic Records**
- Grade management
- Subject-wise marks recording
- Student performance tracking
- Grade distribution
- Academic remarks and feedback

### 👥 **Staff Directory**
- Staff information management
- Designation and department tracking
- Contact information
- Employment status monitoring
- Staff attendance (future integration)

### 📈 **Reports & Analytics**
- Enrollment reports
- Attendance analytics
- Fee collection insights
- Academic performance analysis
- Staff distribution reports
- Export functionality (PDF/Excel)

## Installation & Usage

1. **Open the application:**
   - Open `index.html` in any modern web browser
   - The application will load immediately with sample data

2. **Navigation:**
   - Use the sidebar menu on the left to navigate between modules
   - Click module cards on the dashboard for quick access
   - Page title updates dynamically based on selected module

3. **Responsive Design:**
   - Desktop: Full sidebar with expanded navigation
   - Mobile: Collapsible sidebar for better screen usage

# nizamix-erp
Multi-tenant school ERP for Pakistani schools — student records, attendance, fees, and communication in one platform. Built on Supabase/Postgres.

## Web Demo
A simple `index.html` demo is included for adding and listing students via Supabase.

## Notes
- The current demo uses a Supabase client and requires replacing `PASTE_SCHOOL_ID_HERE` with a valid school ID.
- For production, secure your Supabase keys and move logic to a proper backend.
